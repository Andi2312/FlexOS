# FlexOS

**FlexOS** is a modular operating system based on Linux (Ubuntu Mini 18.06, upgraded to 20.04). The goal of the project is to create a flexible, minimalist, and customizable OS with container support and its own package manager.

## Features

- **XFCE4** — lightweight and fast desktop environment.
- **UI Theme**: Adapta + custom cursor set.
- **Package Manager `fpm` (Flex Package Manager)**:
  - Install, update, and remove `.fpk` packages.
  - Supports dependencies and remote repositories.
- **Containers `fcm` (Flex Container Module)**:
  - Isolated environments for applications and system components.
  - Support for snapshots and rollbacks.
- **Modular Structure**:
  - System and user components work as independent blocks.
  - Allows updates without rebooting the entire system.

## In Development

- Web interface for system management.
- Support for user profiles and custom builds.
- Automatic backups and rollbacks during package installations.

## Goal

To create a lightweight, reliable, and fully customizable OS suitable for both advanced users and beginners.
